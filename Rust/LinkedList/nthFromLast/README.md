# N<sup>th</sup> from last node
[solution](solution.md)

Given a singly linked list, return the n<sup>th</sup> node from the end. Return `NULL` if n is out-of-bounds.

## Example
```
n =     (5)  (4)   (3)   (2)   (1)  (>=6 or <=0)
(head)->[7]->[14]->[21]->[28]->[9]->(NULL)
```
